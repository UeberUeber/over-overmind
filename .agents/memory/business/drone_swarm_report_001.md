## 타겟 고객사 프로파일 (에이전시가 영업할 대상 = 우리가 리서치해줄 회사)

### 🎯 회사 1: 채널톡 (Channeltalk)
| 항목 | 내용 |
|------|------|
| **사업** | AI 기반 고객 메신저 / CRM SaaS |
| **규모** | 고객사 15만+, 연매출 245억원(2024), ARR 360억(2023) |
| **최신 뉴스** | 2025년 카카오와 파트너십 체결, 북미 시장 타겟 마케팅 본격화. AI가 반복 업무를 처리하고 CX 관리자는 고차원 업무에 집중하는 방향으로 솔루션 고도화 중. 2024년 글로벌 진출을 위해 BI(로고) 전면 개편 |
| **콜드메일 훅** | "채널톡이 북미 시장 공략을 본격화하고 있다는 뉴스를 봤습니다. 북미 B2B 타겟 기업 50곳의 CX 스택과 메신저 도입 현황을 리서치해서 맞춤 영업 리스트를 만들어 드릴 수 있습니다." |

### 🎯 회사 2: 센드버드 (Sendbird)
| 항목 | 내용 |
|------|------|
| **사업** | 커뮤니케이션 API → 옴니채널 AI 에이전트 플랫폼으로 전환 |
| **규모** | 글로벌 유니콘급 |
| **최신 뉴스** | 2025년 음성 AI 에이전트 제품군 출시, AWS와 3년 전략적 협약 체결. 2026년 1월 AI 에이전트 간 신뢰성 검증 솔루션 개발 발표. 고객 서비스 자동화를 넘어 "오케스트레이션(Orchestration)" 영역으로 확장 중 |
| **콜드메일 훅** | "센드버드가 AI Agent 오케스트레이션으로 피봇하고 있다는 기사를 읽었습니다. 현재 AI Agent 도입을 검토 중인 글로벌 SaaS 기업 30곳의 기술 스택과 도입 시점을 분석해서 영업 타겟 리스트를 제공해 드릴 수 있습니다." |

### 🎯 회사 3: 리얼월드 (Realworld)
| 항목 | 내용 |
|------|------|
| **사업** | 피지컬 AI (제조/물류/의료 로봇) |
| **규모** | 시드 단계, 누적 투자 600억원 |

| **콜드메일 훅** | "리얼월드의 390억 시드2 투자 유치와 글로벌 실증 진출 소식 축하드립니다. 북미·일본 타겟 시장에서 피지컬 AI/로보틱스 솔루션을 도입했거나 검토 중인 물류·제조 기업 리스트를 맞춤 리서치해 드릴 수 있습니다." |
