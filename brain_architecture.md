# The Brain (Central Overmind Core)

## 목적 (Purpose)
이 모듈은 오버마인드 시스템의 가장 핵심적인 "인지 및 의사결정 엔진"입니다. 외부 자극(데이터)이나 사용자 요청이 들어왔을 때, 기계적으로 파싱하는 것을 넘어 **"이 요청의 의도가 무엇인가?", "현재 우리 시스템(군단)의 상태로 이를 어떻게 처리해야 가장 최적의 진화를 이룰 수 있는가?"**를 스스로 판단하고 지시를 내립니다.

## 주요 기능 (Core Functions)

1.  **의도 파악 및 조율 (Orchestration & Intent Parsing)**
    *   단순 함수 호출이 아닌, LLM의 추론 능력 자체를 래핑(Wrapping)하여 모든 입력의 첫 번째 진입점(Entry Point)이 됩니다.
    *   입력 데이터에 따라 어떤 하위 기관(파서, 저장소, 워커)을 동작시킬지 결정합니다.

2.  **기억(Context) 연동 (Memory Integration)**
    *   과거의 `회고_및_진화` 워크플로우를 통해 얻은 로컬 마크다운 파일(유전자/규칙)들을 읽어 들이고 현재 상황에 적용합니다.

3.  **자율적 목표 생성 (Autonomous Goal Generation)**
    *   사용자가 명령하지 않아도, 빈 시간(Idle)이나 데이터가 축적된 상황에서 스스로 "다음 단계 로직 수정"이나 "데이터 추가 수집" 등의 태스크를 생성할 권한을 가집니다.

## 구현 방향성 (Implementation Plan)
*   Brain은 일종의 '메인 루프(Main Loop)'를 가진 Python 또는 Node.js 스크립트로 구성될 확률이 높습니다.
*   가장 처음 구축할 형태: **사용자의 입력을 받아 `.agents/workflows` 안의 규칙들을 모두 읽은 뒤, 스크립트가 스스로 "다음에 어떤 도구/파일을 수정할지" 결론을 내는 텍스트 기반 추론 엔진.**
